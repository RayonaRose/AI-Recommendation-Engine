
# AI Recommendation Engine

A modern, serverless web application built with **Node.js**, **Express**, and deployed on **Vercel**. This tool provides personalized recommendations across various categories based on user preferences.

## 🚀 Live Demo
* **Live Website:** [Check out the live app here](https://ai-recommendation-engine-beta.vercel.app)

## 🛠️ Built With
* **Backend:** Node.js, Express
* **Environment & Routing:** Vercel Serverless Functions, dotenv
* **Frontend:** HTML, CSS, JavaScript

## ✨ Key Features
* **Dynamic Recommendation Logic:** Processes user categories and inputs to deliver tailored suggestions instantly.
* **Optimized Serverless Architecture:** Configured to run seamlessly as a modern cloud deployment using Node.js modules.
* **Secure Environment Configuration:** Utilizes environment variables to keep sensitive credentials safe and isolated.
